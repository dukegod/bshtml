# responsive web design

`scss` is a amazing css processor, let make css magic!

aim to cross browser, display fluid layout between mobile with pc.


+ setting : 默认设置
    - color : 默认颜色值
    - size 
    - variables : 默认变量
    - default 默认属性的去除
    - breakpoint : 栅栏布局的分割点
+ mixin: 各种mixin
    - grid : 栅栏布局设计
+ function: scss功能函数
    - strip-unit : 返回单位数值
+ components: 封装的组件
+ utilities: 框架对外输出的样式
